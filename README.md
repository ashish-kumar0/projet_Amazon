# 🛒 Amazon Clone — Frontend UI

A pixel-faithful frontend clone of the Amazon India homepage, built entirely with **HTML**, **CSS**, and **JavaScript**. This project replicates the core look and feel of Amazon's interface including a responsive product listing page and a user login/signup flow.

---

## 📸 Preview

> Clone the repo and open `index.html` in your browser to see it live.

---

## 🔗 Repository

```
https://github.com/ashish-kumar0/projet_Amazon.git
```

---

## 🚀 Features

### 🏠 Homepage / Product Listing
- Amazon-style **navbar** with search bar, cart icon, and account section
- **Hero banner** with promotional images
- **Product cards** organized in category sections (similar to Amazon's grid layout)
- **Footer** replicating Amazon's multi-column footer with links

### 🔐 User Login / Auth UI
- Login page styled to match Amazon's sign-in form
- Input fields for **email/phone** and **password**
- "Continue" / "Sign In" button with Amazon-like styling
- Link to a **Create Account** page
- Form UI only — no backend authentication

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **HTML5** | Page structure and semantic markup |
| **CSS3** | Styling, layout (Flexbox/Grid), responsive design |
| **JavaScript (Vanilla)** | DOM interactions, dynamic behavior |

> No frameworks. No libraries. Pure vanilla frontend.

## ⚙️ How to Run Locally

No setup, no installs. Just:

```bash
# 1. Clone the repository
git clone https://github.com/ashish-kumar0/projet_Amazon.git

# 2. Navigate into the folder
cd projet_Amazon

# 3. Open the homepage in your browser
open index.html      # macOS
start index.html     # Windows
xdg-open index.html  # Linux
```

Or simply double-click `index.html` from your file explorer.
